#  SQL Assignment – Hotel & Clinic Management System

##  Overview

This project is part of a data analysis assignment. It focuses on designing relational databases and writing SQL queries for two real-world systems:

* Hotel Management System
* Clinic Management System

The project demonstrates table creation, data insertion, and querying structured data.

---

##  Technologies Used

* SQL (MySQL / SQLite)
* VS Code

---

##  Files Included

* hotel_management.sql → Hotel database schema and queries
* clinic_management.sql → Clinic database schema and queries
* README.md

##  Hotel Management System

### Tables Created

### Operations Performed
